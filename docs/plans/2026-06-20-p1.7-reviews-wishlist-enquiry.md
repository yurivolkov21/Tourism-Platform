## Risks / gotchas
- **`@nestjs/throttler` must be a direct dep** or boot fails under webpack externalization (smoke catches).
- **Decorated controller return types** → `import type` (TS1272).
- **TS6305 flood** on typecheck = stale buildinfo → `cd apps/api && rm -rf dist && pnpm exec tsc -b
  tsconfig.json --emitDeclarationOnly --force`.
- **Pooler:** keep list+count on `Promise.all` (no `$transaction`); only avatar set / (future) uses tx.
- Donor `wishlist`/`admin-stats` carry `*_vi`/`titleEn`/`destinationId` — **must** adapt or typecheck fails.
