# P1.4c — Tour departures · implementation plan

**Spec:** [../06-specs/2026-06-16-p1.4c-departures-design.md](../06-specs/2026-06-16-p1.4c-departures-design.md)
**Branch:** `feat/p1.4c-departures` · port source: donor `modules/departures` + P1.4b pattern.

Dependency-ordered. TDD on service logic (date/seat guards, public-vs-admin reads).
Run `/gate` (api scope) + smoke before green. Decimal money → validate `>=0`, serialize as string.

## Tasks

1. **DTOs** (`modules/departures/dto/`):
   - `create-departure.dto.ts` — `startDate`/`endDate` (strict ISO-8601), `seatsTotal` (1..1000),
     `priceOverride?` + `compareAtPrice?` (Decimal, Min 0), `status?` (`DepartureStatus`).
     `seatsBooked` deliberately absent.
   - `update-departure.dto.ts` — `PartialType(CreateDepartureDto)` (null clears price fields via `@IsOptional`).
   - `list-departures-query.dto.ts` — `from?`/`to?` (ISO-8601) + `status?`.
   - `departure.dto.ts` — response shape incl. `priceOverride` + `compareAtPrice` (string|null), `seatsBooked`.
   - ✅ `nx typecheck @tourism/api` green.

2. **DeparturesService** (`departures.service.ts` + spec) — TDD, port donor + `compareAtPrice`:
   - `findPublicListForTour(slug, q)` — published-tour gate, default `from=today`/`status=OPEN`, asc.
   - `findAdminListForTour(slug, q)` — no gate, literal filters, asc.
   - `create(slug, body)` — tour 404, INVALID_DATE_RANGE, DEPARTURE_IN_PAST, seatsBooked=0, Decimals.
   - `update(slug, id, body)` — 404s, partial date guard, SEATS_TOTAL_BELOW_BOOKED, null-clear price fields.
   - `remove(slug, id)` — DEPARTURE_HAS_BOOKINGS (seatsBooked>0) + `P2003` fallback.
   - helpers: `findTourBySlugOrThrow` / `findPublishedTourBySlugOrThrow` / `findDepartureOrThrow`,
     `assertDateRange`, `buildDateFilter`, `startOfToday`, `mapUpdatePayload`.
   - ✅ spec ~10 cases green.

3. **Controllers + module + wiring**:
   - `departures.controller.ts` (`@Public`, `/tours/:slug/departures`) — list.
   - `admin-departures.controller.ts` (`@Roles(ADMIN)`, `/admin/tours/:slug/departures`) — list/create/
     patch(`:id`, `ParseUUIDPipe`)/delete(`:id`).
   - `departures.module.ts` (export `DeparturesService` for P1.5 bookings); register in `app.module.ts`.
   - ✅ app boots; routes mapped.

4. **Gate** — `pnpm nx run-many -t lint typecheck test build --projects=@tourism/api`. ✅ green.

5. **Smoke** — build → kill stale :3000 → boot: `GET /tours/nope/departures` → 404 envelope;
   `GET /admin/tours/nope/departures` (no token) → 401. ✅ enveloped.

6. **PR** — commit (conventional), push, PR, **rebase-merge + delete branch** (remote `git push
   origin --delete` + verify). Update `docs/roadmap.md` (P1.4 ✅ complete) + `.remember`.
