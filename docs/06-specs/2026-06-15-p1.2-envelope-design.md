# P1.2 — Response envelope + security bootstrap · design spec

**Date:** 2026-06-15 · **Branch:** `feat/p1.2-envelope` · **Phase:** P1.2
**Refs:** [backend.md](../01-architecture/backend.md) · ADR-0008 (security) · donor `common/`

## Goal

The cross-cutting backend foundation every later module builds on: a uniform
response envelope, a global exception filter, and a hardened bootstrap. Ported
from the donor, adapted to Nx + EN-only + the shared-types boundary.

## In scope

- **`@tourism/core`**: shared `ApiResponse<T>` / `ApiError` / `ApiMeta` types
  (consumed by api now, FE later). First `scope:api → scope:shared` link.
- **apps/api `common/`**: `TransformInterceptor` (success → envelope),
  `HttpExceptionFilter` (failure → envelope, 5xx stack hidden), `@SkipTransform`.
- **Bootstrap (`main.ts`)**: dotenv (robust path), Sentry init (guarded by DSN),
  helmet, CORS allowlist, global prefix, global `ValidationPipe`
  (whitelist + transform), Swagger (non-prod).
- Global wiring via `APP_INTERCEPTOR` + `APP_FILTER`.
- Webpack: externalize node_modules (pnpm app-deps fix).

## Out of scope (later)

- Auth guards/decorators (`@CurrentUser`/`@Public`/`@Roles`) → **P1.3**.
- `express.raw()` for payment webhooks/IPN → **P1.5** (paths known then).
- Structured logging (pino) → later; Nest `Logger` for now.

## Envelope shape

`{ data: T | null, error: ApiError | null, meta?: ApiMeta }`. Interceptor
branches: opt-out (`@SkipTransform`) · already-shaped passthrough ·
`{ items, meta }` → hoist · plain value → wrap.

## Decisions / notes

- **MoMo env keys optional until P1.5** (relaxed Joi; tighten to `.required()`
  when payments land) — otherwise the empty placeholders block boot.
- **Webpack externalizes all bare imports** (`mergeExternals` + custom externals):
  Nx's `nodeExternals` scans ROOT `node_modules`, but pnpm puts app deps under
  `apps/api/node_modules`; bundling them broke on `@nestjs/mapped-types`'
  guarded `require('class-transformer/storage')` (false positive — runtime uses
  `cjs/storage`).

## Verification

`prisma`-independent unit tests (interceptor 5, filter 4, core 3) + gate green +
**runtime smoke**: app boots, `PrismaService` connects to the new Supabase DB,
`GET /api/v1` returns `{data,error:null}`, `/api/docs` → 200.
